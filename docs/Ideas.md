# Ideas

## Question

・社会とは何か。

・制度とは何か。

・コミュニティとは何か。

・ゲームとは何か。

・AIとは何か。

・人とは何か。

## 2026-07-06

・制度に寿命がくる。人に寿命があるように。

・AIにも寿命がくる。人に寿命があるように。

・コミュニティにも寿命がくる。人に寿命があるように。

・参加者自身も制度の対象となる。

・エントロピーは数値ではなく見た目で感じられるようにしたい。

・社会実装の前段階のゲームとして。

## 2026-07-07

仕事で自己評価を書いていて違和感を覚えた。

人を評価するより、
社会の循環を評価する方が自然ではないか。

## 2026-07-07

金融工学の話を聞いて、
経済や社会には突然のショックが起こることを考えた。

Self Sunset Protocol では、
強制介入ではなく、
ショックを循環の力で受け流し、
分散し、発散させる仕組みにしたい。

## 2026-07-08

ニュースで国会に人が集まり、中央で物事を決める構造を見て、
中央集権そのものに違和感を覚えた。

Self Sunset Protocol では、
権利や決定権を一つの中心に集中させず、
役割に応じて自動的に分散させたい。

中央が存在する場合でも、
それは永続的な支配者ではなく、
寿命を持つ一つの役割として扱う。

## 2026-07-08

義務教育の範囲について考えた。

教育は大事だが、
学校に集まることだけを前提にしなくてもよいのではないか。

Self Sunset Protocol では、
年齢ではなく役割に応じて学ぶ社会を考えたい。

役割を受け取ると同時に教育も行われ、
人生のどの段階でも参画できる仕組みにしたい。


## 会話と主義の距離

仕事の会議で、自然な会話や雑談に近い話をしようとしたとき、
「なぜその話をしたのか」と問われた。

その問い自体は合理的かもしれない。

しかし、
すべての会話に理由や成果を求められると、
人は会話すること自体をためらうようになる。

Self Sunset Protocol では、
会話を成果や合理性だけで扱わず、
余白、雑談、違和感、感情、まだつながっていない点も、
コミュニティの循環に必要なものとして扱いたい。
