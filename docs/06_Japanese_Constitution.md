# Constitutional View

## 日本国憲法との向き合い方

Self Sunset Protocol は、日本国憲法を否定するためのものではありません。

このプロジェクトは、現行憲法が守っている個人の尊重、基本的人権、平和、地方自治、権力の制限といった価値を踏まえながら、
社会や制度がどのように新陳代謝できるかを考えるための実験です。

Self Sunset Protocol は、現実の法制度を置き換えるものではなく、
制度設計を考えるためのゲーム・シミュレーション・公開研究ノートです。

## このプロジェクトの位置づけ

## 尊重する価値

Self Sunset Protocol は、次の価値を重視します。

- 個人が尊重されること
- 権利が一部の人に独占されないこと
- 権力が固定化しないこと
- 参加者が人生のどの段階でも学び、参画できること
- 地域やコミュニティが自律的に循環できること
- AIや制度も見直しの対象になること

## 日本国憲法と Self Sunset Protocol の対応

| 日本国憲法の視点 | Self Sunset Protocol で考えること |
|---|---|
| 個人の尊重 | 人を評価対象ではなく、循環の担い手として扱う |
| 基本的人権 | 権利を中央に集中させず、役割と参加に応じて開く |
| 国民主権 | 固定された代表だけでなく、担い手が循環から浮上する |
| 権力分立 | 権限、制度、AI、知識を分散させ、寿命を持たせる |
| 地方自治 | 小さなコミュニティが自律的に循環できる構造を考える |
| 改正 | 制度そのものにも寿命を持たせ、見直しを前提にする |

## 変えてよいもの、変えてはいけないもの

Self Sunset Protocol では、
役割、制度、AI、権限、運用方法には寿命を持たせます。

しかし、
個人の尊重や基本的人権のような価値は、
簡単に失効させる対象として扱いません。

変えるべきものは、権力の形です。

守るべきものは、人の尊厳です。

## 今後考える問い

- 中央集権を弱めることと、公共の安定をどう両立するか
- 投票を減らすことと、民主的な正当性をどう両立するか
- AIが候補を浮上させる仕組みと、人間の自由意思をどう守るか
- 役割に応じた権利の自動付与と、法の下の平等をどう両立するか
- 制度に寿命を持たせることと、基本的人権の安定性をどう分けるか、制度には寿命を持たせても、人権そのものを寿命付きにしてはいけない
