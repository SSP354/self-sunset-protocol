# Principles

## 評価

Self Sunset Protocol は、
個人を評価することを目的としません。

評価の対象は個人ではなく、
社会全体の循環や承継、
知識の流れとします。

個人を点数で比較するのではなく、
仕組みが健全に機能しているかを観察します。

人は評価の対象ではなく、循環の担い手である。

## 分散された権利

Self Sunset Protocol では、
権利や決定権を一つの中心に集中させません。

参加者は、役割を持った時点で、
その役割に必要な権利と責任を自動的に受け取ります。

権利は上から与えられるものではなく、
役割、参加、承継、循環の中で発生するものとして扱います。

中央の権力がすべてを決めるのではなく、
それぞれの場所で必要な判断が行われ、
その判断は記録され、見える化され、必要に応じて見直されます。

Self Sunset Protocol では、
権利を集中させるのではなく、
役割に応じて分散させます。

中央は存在してもよい。

ただし、
中央は永続してはならない。

中央も役割であり、
寿命を持ち、
既得権益指数とAI統制指数の観察対象となる。

## 年齢ではなく役割に応じて学ぶ

Self Sunset Protocol では、
教育を特定の年齢や場所だけに閉じ込めません。

学ぶことは、学校に集まることだけで行われるものではありません。

参加者は、役割を受け取ると同時に、
その役割に必要な知識、経験、背景、判断の理由を学びます。

教育は、役割と分離されたものではなく、
承継の一部として扱われます。

年齢によって参加や学習の機会を閉じるのではなく、
人生のどの段階でも、
学び、担い、手放し、次へ渡すことができる社会を目指します。

Self Sunset Protocol では、
教育とは一度終わるものではなく、
一生を通じて循環するものです。


## 裁きではなく、回復と責任の循環を設計する

Self Sunset Protocol では、
加害行為が起きたとき、
被害者の感情や人生への影響を制度の中心に置きます。

被害者は、
自分に起きた被害、失われた可能性、必要な回復条件、
加害者に求める説明や制限を表明できます。

加害者は、
その内容を理解し、説明し、責任を引き受け、
必要であれば再対話や再審を求めることができます。

AIや技術は、
被害の可視化、感情の整理、人生分岐の記録、
対話の補助に使われます。

しかし、AIや技術は裁判官ではありません。

また、被害者の感情も、
そのまま刑罰の重さに変換されるべきではありません。

Self Sunset Protocol では、
裁きを一度きりの決定として扱うのではなく、
被害回復、責任、再発防止、対話、見直しが続く循環として扱います。


### 安全装置

この仕組みでは、
被害者、加害者、AI、第三者のいずれか一つに、
最終決定権を集中させません。

被害者の声は中心に置かれますが、
報復や過度な刑罰を防ぐために、
手続、記録、再対話、異議申し立て、第三者確認を必要とします。

加害者にも、
説明、反論、再審、再対話の機会を残します。

AIは可視化と整理を行いますが、
刑罰や処遇を自動決定してはなりません。


## 負の循環を修復循環へ戻す

Self Sunset Protocol では、
加害行為によって負の循環が生まれることを前提とします。

負の循環とは、
被害、怒り、恐怖、報復、孤立、沈黙、再加害が連鎖し、
社会やコミュニティの循環が傷ついていく状態です。

加害行為が起きたとき、
重要なのは、被害をなかったことにすることではありません。

また、加害者を単純に排除することだけでもありません。

Self Sunset Protocol では、
まず被害を記録し、
被害者の安全を確保し、
加害者の責任を明確にし、
再発防止の条件を設定します。

そのうえで、
負の循環がさらに広がらないように、
対話、距離、制限、教育、見直し、修復循環を組み合わせます。

目的は、
罪を軽く扱うことではありません。

被害を中心に置きながら、
報復、沈黙、孤立、再加害の連鎖を止めることです。

Self Sunset Protocol では、
負の循環を放置せず、
記録、責任、回復、再発防止を通じて、
可能な限り修復循環へ戻すことを目指します。


## 主義からの独立性

Self Sunset Protocol は、
特定の経済思想、政治思想、社会思想に支配されることを目的としません。

資本主義、社会主義、国家主義、技術主義、AI主義など、
どのような主義であっても、
それが固定化し、制度や人の循環を妨げる場合、
Self Sunset Protocol の観察対象となります。

Self Sunset Protocol は、
ある主義を別の主義で置き換えるためのものではありません。

重要なのは、
どの主義が正しいかではなく、
人、制度、資源、知識、役割が健全に循環しているかです。

そのため、
特定の主義が社会の上位に固定され、
食料、住居、教育、役割、情報への参加を妨げる場合、
その状態を既得権益化として観察します。

Self Sunset Protocol では、
主義を支配者にしません。

主義もまた、
必要に応じて見直され、
寿命を持ち、
循環の対象として扱われます。


## 生存基盤を循環させる

Self Sunset Protocol では、
食料、住居、教育、医療、情報、役割への参加を、
人が生きるための生存基盤として扱います。

生存基盤は、
投機、独占、固定化によって閉じられるべきではありません。

食料は、
生産、消費、廃棄、再利用、再分配の循環として扱います。

住居は、
所有だけでなく、
選択、利用、共有、放棄、再配分の循環として扱います。

誰かが永久に占有するためではなく、
必要な人が必要な時期に使い、
役割や人生の変化に応じて手放し、
次の人へ渡せる仕組みを考えます。

Self Sunset Protocol では、
生存基盤を市場だけに委ねず、
循環の状態として観察します。
