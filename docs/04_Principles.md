# Principles

## 評価

Self Sunset Protocol は、
個人を評価することを目的としません。

評価の対象は個人ではなく、
社会全体の循環や承継、
知識の流れとします。

個人を点数で比較するのではなく、
仕組みが健全に機能しているかを観察します。

人は評価の対象ではなく、循環の担い手である。

## 分散された権利

Self Sunset Protocol では、
権利や決定権を一つの中心に集中させません。

参加者は、役割を持った時点で、
その役割に必要な権利と責任を自動的に受け取ります。

権利は上から与えられるものではなく、
役割、参加、承継、循環の中で発生するものとして扱います。

中央の権力がすべてを決めるのではなく、
それぞれの場所で必要な判断が行われ、
その判断は記録され、見える化され、必要に応じて見直されます。

Self Sunset Protocol では、
権利を集中させるのではなく、
役割に応じて分散させます。

中央は存在してもよい。

ただし、
中央は永続してはならない。

中央も役割であり、
寿命を持ち、
既得権益指数とAI統制指数の観察対象となる。
