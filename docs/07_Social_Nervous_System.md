# Social Nervous System

## 社会神経網

Self Sunset Protocol では、
社会を数年に一度だけ意思を示す存在として扱わない。

人が日常生活の中で生じた疑問、違和感、
不安、困難、希望、発見を、
必要な瞬間に社会へ返せる仕組みを持つ。

これらの入力は、
社会の状態を示す信号として扱われる。

社会入力端末は、
家庭、職場、公共空間、移動先など、
世界のどこからでも利用できる。

ただし、
端末は人の生活を常時観察するものではない。

人が自ら入力した瞬間だけ、
社会との接点が生まれる。


## 社会信号の流れ

```text
日常の疑問・違和感
        ↓
社会入力端末
        ↓
元の信号を保存
        ↓
個人情報の最小化
        ↓
類似信号・反対信号との接続
        ↓
地域・時間・影響範囲の確認
        ↓
社会の波として可視化
```



## 地球全体の波

```md
## 地球全体の波

社会信号は、
単純な賛成数や反対数としてだけ表示しない。

地球または地域の地図上に、
問題の広がり、深刻度、緊急性、
継続時間、不確実性を
波として視覚化する。

波の大きさは、
件数だけで決めない。

生命への危険、
権利侵害、
回復困難性、
影響の集中、
時間的緊急性を別々に表示する。

一人から発せられた重大な信号も、
多数の軽い要望に埋もれないようにする。
```

## 社会入力端末の条件

社会入力端末は、次の条件を持つ。

- 疑問だけでも入力できる
- 短い文章または音声で利用できる
- 入力しない自由を保障する
- 位置情報の送信を標準としない
- 広告や政治宣伝を表示しない
- 元の入力とAIの要約を区別する
- 入力後の処理経路を確認できる
- AIによる分類に異議を申し立てられる
- 不正が疑われた場合に交換・停止できる
- 単一企業または単一国家だけが管理しない
