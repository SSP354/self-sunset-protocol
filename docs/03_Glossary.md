# Glossary

## 循環評価

循環評価とは、個人の能力や価値を点数化するのではなく、
コミュニティ全体の流れが健全に機能しているかを観察する考え方です。

Self Sunset Protocol では、
評価の対象を個人ではなく、
権限、知識、役割、情報、参加の循環に置きます。

人は評価される対象ではなく、
循環を担う存在として扱われます。

## 承継圧縮

承継圧縮とは、
過去の知識を失わずに、
次の参加者が受け取りやすい形へ整理する仕組みです。

## AI観察者

AI観察者とは、
個人を評価するのではなく、
社会やコミュニティの循環状態を観察するAIです。

AI観察者は支配者ではなく、
承継と可視化を助ける補助者です。

AI観察者にも寿命があり、
定期的に見直されます。

## 担い手の浮上

担い手の浮上とは、
投票や立候補だけに頼らず、
役割、知識、情報、参加の循環から、
次の役割を担う可能性のある参加者が自然に見えてくる仕組みです。

Self Sunset Protocol では、
次の担い手は固定された人間関係から選ばれるのではなく、
循環の流れの中から浮上するものとして扱われます。

## 既得権益指数

既得権益指数とは、
権限、知識、役割、情報、参加の流れが、
どれだけ固定化しているかを観察するための指標です。

Self Sunset Protocol では、
既得権益指数を個人を責めるためには使いません。

社会やコミュニティの循環が硬くなっている場所を見つけ、
改善するために使います。

## 制度の寿命

制度の寿命とは、
制度が自動的に見直し対象となるまでの期間です。

Self Sunset Protocol では、
制度は永遠に残るものではなく、
必要な期間だけ社会やコミュニティを支える仕組みとして扱われます。

寿命を迎えた制度は、
継続、修正、統合、停止、Archiveへの移動のいずれかを選びます。

## AIの寿命

AIの寿命とは、
AIが自動的に見直し対象となるまでの期間です。

Self Sunset Protocol では、
AIを永続的な支配者として扱わず、
一定期間だけ循環を支える観察者・補助者として扱います。

寿命を迎えたAIは、
継続、更新、分割、権限縮小、停止、Archiveへの移動のいずれかを選びます。


## 循環的発散

循環的発散とは、
社会やコミュニティに発生した衝撃を、
強制的な介入や一方的な排除によって処理するのではなく、
役割、知識、制度、AI、参加者の循環を通じて分散させる考え方です。

Self Sunset Protocol では、
稲妻を完全になくすことはできないと考えます。

重要なのは、
衝撃を一部の人や制度に集中させず、
社会全体の流れの中で受け流し、
次の更新へつなげることです。

## 承継教育

承継教育とは、
役割を担う参加者が、
その役割に必要な知識、経験、判断の理由を受け取りながら学ぶ仕組みです。

Self Sunset Protocol では、
教育を年齢や学校だけに限定せず、
役割の承継とともに発生するものとして扱います。

承継教育は、
参加者が人生のどの段階でも学び、
役割を担い、
次の参加者へ知識を渡せるようにするための考え方です。


## 修復循環

修復循環とは、
加害行為によって生じた被害を、
一度きりの刑罰だけで終わらせるのではなく、
被害の可視化、加害者の理解、責任の確認、
回復条件、再発防止、再対話、見直しを通じて扱う仕組みです。

修復循環では、
被害者の声を制度の中心に置きます。

しかし、
被害者、加害者、AI、第三者のいずれか一つに、
最終決定権を集中させません。

Self Sunset Protocol では、
裁きもまた固定された一回限りの決定ではなく、
回復と責任の循環として考えます。


## 永続記録と再検証

永続記録と再検証とは、
時間が経過しても、
被害、加害、判断の経緯、未解決の問いを記録として残し、
必要に応じて再び検証できるようにする考え方です。

Self Sunset Protocol では、
時効を単なる忘却として扱いません。

新しい証拠、新しい技術、新しい被害との関連が見つかった場合、
過去の記録は再び開かれます。

ただし、
記録が残ることと、
処罰が永遠に可能であることは同じではありません。

記録は、
復讐のためではなく、
被害の理解、再発防止、修復循環、社会の承継のために残されます。


## 主義からの独立性

主義からの独立性とは、
Self Sunset Protocol が特定の経済思想、政治思想、社会思想に支配されないことです。

資本主義、社会主義、国家主義、技術主義、AI主義など、
どのような主義であっても、
固定化し、循環を妨げる場合は観察対象となります。

Self Sunset Protocol は、
主義を否定するためではなく、
主義が制度や人の循環を支配しないようにするために存在します。

## 生存基盤

生存基盤とは、
人が生き、学び、参加し、役割を担うために必要な基盤です。

食料、住居、教育、医療、情報、役割への参加などが含まれます。

Self Sunset Protocol では、
生存基盤を投機や独占の対象としてだけ扱うのではなく、
社会全体で循環させる対象として考えます。


## 伴走型会話支援

伴走型会話支援とは、
会話の中で感情が高まったり、
相手との距離感が難しくなったりしたときに、
AIが参加者の内側に寄り添い、
言葉、感情、距離感を整える補助を行う仕組みです。

伴走型会話支援は、
会話を監視するためのものではありません。

また、
相手を評価したり、
会話を支配したりするためのものでもありません。

Self Sunset Protocol では、
AIを会話の裁判官としてではなく、
会話の波を整える伴走者として扱います。


### 負の循環

一つの問題が次の問題を生み、
その結果がさらに元の問題を強化していく連鎖。

例：
孤立 → 生活不安 → 問題行動 → 排除 → さらなる孤立


### 循環遮断

負の循環が拡大する前に、
役割、権限、接触、制度運用などを一時的に停止または制限し、
連鎖の拡大を防ぐこと。


### 再接続

責任を果たし、再発防止や更生が確認された人を、
社会的役割、仕事、地域、組織などの循環へ再び接続すること。


### 再接続義務

社会や組織が、更生した人に対して再参加の経路を完全に閉ざさず、
能力、安全性、状況に応じた参加の機会を持つ責任。


### 再循環

問題や犯罪によって社会との接続を失った人が、
再び社会の中で役割を持ち、安定した循環に戻ること。


## 社会入力端末

社会入力端末とは、
参加者が日常生活の中で生じた疑問、違和感、
不安、希望、発見を、
社会信号として容易に入力するための共通端末です。

端末は、複雑な政治的文章や政策案の作成を
参加者に要求しません。

短い文章、音声、選択肢などによって、
必要最小限の入力ができることを重視します。

社会入力端末は、
利用者の生活を常時監視する装置ではありません。

本人が明示的に入力した情報だけを扱い、
位置情報、会話、生体情報、私生活を
自動的に収集してはなりません。


## 疑問信号

疑問信号とは、
明確な意見や解決案になる前の、
「なぜこうなっているのか」
「これはおかしくないか」
「このままで安全なのか」
という未整理の問いです。

Self Sunset Protocol では、
完成した主張だけでなく、
まだ答えのない疑問も社会信号として扱います。

## 一時決定循環

一時決定循環とは、
特定の社会的課題に対応するため、
当事者、専門家、一般参加者、
必要に応じたAI支援によって
一時的に構成される意思決定単位です。

一時決定循環には、
目的、権限、予算、期間、停止条件、
修正条件および解散条件が設定されます。

課題の終了または期限の到来によって、
その権限は社会の循環へ返却されます。

## 即時修正

即時修正とは、
決定や施行の結果に重大なずれ、
被害、予期しない影響が確認された場合に、
任期や見直し日を待たず、
決定内容、権限、資源配分を
一時停止または変更できる仕組みです。

即時修正は、
一人の判断で無制限に行うものではなく、
事前に定めた条件と記録に基づいて行います。


## 設計者可視化

設計者可視化とは、
候補者の浮上条件、優先順位、
AIの分類基準、意思決定手続などを
誰が設計し、変更したのかを見える状態にすることです。

制度の利用者だけでなく、
制度の設計者、更新者、データ提供者も、
循環評価と既得権益指数の観察対象となります。


## 権限の離散

権限の離散とは、
一時決定循環が役割を終えたあと、
その権限を別の恒久的な権力者へ移すのではなく、
細分化し、失効させ、
社会全体の循環へ返すことです。


## 民意制約

民意制約とは、
社会信号から形成される、
解決策が満たすべき条件と、
越えてはならない境界です。

民意制約は、
具体的な担当者や方法を先に決めるものではありません。

生命、安全、権利、予算、時間、
地域への影響などについて、
一時的な決定循環が守るべき条件を示します。
