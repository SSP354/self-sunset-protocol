## 日本国憲法との問い

Self Sunset Protocol は、日本国憲法を否定するものではありません。

しかし、現行の制度や社会の形を見つめ直すために、
次の問いを持ち続けます。

- 権力を固定化させずに、人の尊厳を守るにはどうすればよいか。
- 制度に寿命を持たせながら、基本的人権の安定性をどう守るか。
- 中央集権を弱めながら、社会全体の安全や公共性をどう保つか。
- 投票や代表制に頼りすぎず、参加者の意思や役割をどう反映するか。
- AIを補助者として使いながら、AIが新しい支配者にならないようにするにはどうすればよいか。
- 年齢や所属に縛られず、人生を通じて学び、役割を担い、承継できる社会は作れるか。


## 被害回復と対話の循環

Self Sunset Protocol では、
犯罪や加害行為を単なる法律違反としてだけではなく、
人の人生、役割、関係性、未来の分岐を傷つける出来事として考えます。

被害を受けた本人や親族の感情、喪失、恐怖、怒り、失われた可能性は、
制度の外に置き去りにされるべきではありません。

将来的な技術によって、
被害者の置かれた状況、人生の分岐、失われた道筋、
感情的な影響を可視化できるようになった場合、
それを加害者と共有し、対話する仕組みを考えます。

ただし、
被害者の感情をそのまま刑罰に変換することには慎重でなければなりません。

重要なのは、
復讐を制度化することではなく、
被害の理解、責任の確認、回復条件、再発防止、
そして継続的な対話の機会を設計することです。

Self Sunset Protocol では、
被害者と加害者が一度きりの裁判で分断されるのではなく、
必要に応じて何度でも説明し、確認し、見直し、
納得に近づくための対話の循環を考えます。


## 修復循環の問い

- 被害者の声を、制度の中心にどう置くか。
- 被害者の感情を、復讐ではなく回復へつなげるにはどうするか。
- 加害者に、理解、説明、責任、再対話の機会をどう残すか。
- AIによる被害の可視化が、新しい支配や誤判定にならないようにするにはどうするか。
- 裁きを一度きりの決定ではなく、見直し可能な循環として扱えるか。
