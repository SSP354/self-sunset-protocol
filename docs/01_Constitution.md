# Constitution

## Self Sunset Protocol 憲法

この憲法は、Self Sunset Protocol の基本原則を定めます。

この憲法は完成を目的としません。

必要に応じて見直され、
Self Sunset Protocol とともに成長します。

---

## 第一条　承継

Self Sunset Protocol は、
特定の個人に属するものではありません。

参加者が役割を終えても、
知識、経験、思想は次の世代へ承継されます。

承継されるのは権限ではなく、
社会をより良くしようとする意思です。

## 第二条　人の非失効領域

役割、制度、権限、AIおよび意思決定機関には、
寿命と失効条件を持たせる。

ただし、人の生命、身体、安全、尊厳、
思想、良心、表現、プライバシー、
適正な手続を受ける権利は、
Self Sunset の対象としてはならない。

多数の社会信号、
一時的な緊急性、
制度上の効率、
AIの判断または循環指標を理由として、
人としての基本的な権利を失効させてはならない。

役割に伴う権限は消失する。

人であることに伴う権利は消失しない。

## 第三条　生命と権利の優先

社会信号の件数と、
問題の重大性は同じものではない。

生命、身体、安全、尊厳および基本的人権に
回復困難な影響を与える問題は、
信号の件数が少ない場合であっても、
独立した検討と保護の対象となる。

多数であることは、
少数者の生命や権利を失わせる根拠にはならない。

## 第四条　社会信号への参加

すべての参加者は、
日常生活の中で生じた疑問、違和感、不安、
困難、希望および発見を、
社会信号として表明できる。

社会信号を入力すること、
入力しないこと、
匿名で入力すること、
入力後に修正または撤回を求めることは、
参加者の権利として扱われる。
