# Self Sunset Protocol

## 概要

Self Sunset Protocol は、人・制度・コミュニティが自然に新陳代謝する社会を探究するオープンプロジェクトです。

このプロジェクトでは、

- 権限には必ず終わりがある
- 制度にも寿命がある
- 知識は次世代へ受け継がれる

という考え方をゲームとして体験できる社会シミュレーションを目指します。
